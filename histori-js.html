ECMAScript

É uma especificação da linguagem de programação padronizada, servindo como base futuras implementaçoes de outras linquagens Scripts.
ECMAScript é uma especificação de linguagem script criada pela Ecma International, utilizada por linguagens como: ActionScript, JScript e Javascript. Quem cuida do projeto do ECMAScript é o TC39, é o comiter responsavel pelo javaScript.

Stage 0: strawman, Stage 1: proposal, Stage 2: draft, Stage 3: candidate e Stage 4: finished

spread

permite passa arrays como parametro e tambem passar arrays dentro de outos arrays.

EX: var partes = ['ombro','joelho'];
var musica = ['cabeça', ...partes, 'e', 'pés'];

ísso é o mesmo que : var musica = ['cabeça','ombro','joelho', 'e', 'pés'];
